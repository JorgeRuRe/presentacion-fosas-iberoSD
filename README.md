# Usar datos para desenterrar tragedias - Social Data Ibero 

Este es un repositorio con las diapositivas en formato PDF de la presentación titulada "Usar datos para desenterrar tragedias ¿Cómo se puede utilizar la ciencia de datos para predecir la existencia de fosas clandestinas en México?, impartida el 05 de marzo de 2021 en el seminario de métodos del Social Data Ibero. Los archivos se pueden descargar en zip, haciendo click en el botón de CODE. El PDF se encuentra en la carpeta con el nombre "output" y las gráficas e imágenes utilizadas están en la carpeta de "input". 

La plática puede consultarse en: https://www.youtube.com/watch?v=87ZDNuq2Sgo&feature=youtu.be
